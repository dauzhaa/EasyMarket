# EasyMarket

## Описание
Пет-проект интернет-магазина с полным циклом разработки

## Технологии
- **Backend**: Django 6.0, Django REST Framework
- **Frontend**: React, Typescript
- **База данных**: PostgreSQL
- **Кэш**: Redis
- **Контейнеризация**: Docker (планируется)

## MVP функционал
- Каталог товаров
- Корзина закупок
- Оформленеи заказов
- История заказов
